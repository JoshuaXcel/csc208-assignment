**Name:** Ngullah Mudana Joshua

A Java console program that records student names, courses, units, and scores,

computes letter grades and GPA, and exports a structured report in CSV

(Excel-readable) format.

## How to run

```bash

javac GradingSystem.java

java GradingSystem

```

Output is written to `grading_report.csv`.
